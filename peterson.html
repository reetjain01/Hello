Peterson's Algorithm

Peterson's algorithm is a concurrent programming algorithm for mutual exclusion that allows two or more processes to share a single-use resource without conflict, using only shared memory for communication.

Peterson's solution was proposed to resolve the critical section problem involving only two processes. This solution guarantees that it provides mutual exclusion, bounded waiting, and progress of the processe s.
